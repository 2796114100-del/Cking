产品摄影和美食摄影是商业摄影的重要分支。

一、产品摄影

器材：
- 相机：任何相机都可以，重要的是灯光
- 镜头：90-105mm微距镜头（产品细节）或50mm标准镜头
- 灯光：至少2盏LED灯 + 柔光箱
- 背景：白色/黑色/渐变背景纸或背景板
- 三脚架：必须，保持构图稳定

灯光布置：
- 双灯布光：左右各一盏45度柔光灯，消除阴影
- 顶光：正上方打光，适合透明/反光产品
- 背光：背后打光，适合透明产品（如玻璃杯）
- 反光板：补暗部细节

拍摄技巧：
- 使用手动曝光，保持每张照片曝光一致
- 手动白平衡，确保颜色准确
- 光圈F8-F11，保证产品整体清晰
- 多角度拍摄：正面、侧面、45度、特写细节
- 清洁产品表面，避免指纹和灰尘

常见产品类型：
- 电商产品：白底图 + 场景图
- 珠宝首饰：微距 + 黑底 + 点光源
- 电子产品：展示屏幕和接口细节
- 服装：模特穿着 + 平铺 + 挂拍

二、美食摄影

器材：
- 镜头：50mm或90mm微距
- 灯光：自然光最佳（窗户光），或LED灯+柔光箱
- 道具：餐具、桌布、鲜花、调料瓶

拍摄角度：
- 45度角：最自然，模拟用餐视角
- 俯拍（90度）：适合摆盘丰富的菜品
- 平拍（0度）：适合汉堡、蛋糕等有高度的食物

布光技巧：
- 侧逆光（45度后方）：让食物有光泽和层次
- 避免正面光：食物会显得平、没有质感
- 用反光板补暗部

食物造型：
- 拍摄前在食物表面刷一层油，增加光泽
- 热食要趁热拍（蒸汽效果）
- 用喷壶喷水雾，模拟冷饮的冷凝效果
- 酱汁/蜂蜜的浇淋动作要连拍

后期处理：
- 提高饱和度（食物要鲜艳）
- 调整白平衡（食物偏暖更好看）
- 局部提亮高光（增加光泽感）
- 去除背景杂物
